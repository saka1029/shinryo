|-|-|

# テキストの生成

PDFファイルからテキストファイルを
生成する。

```
./run NN g0
```

# テキストの編集

最初に一度だけ
`in/NN/g/txt/k.txt`
を
`in/NN/g/txt/ke.txt`
にコピーする。
コピー後は`ke.txt`のみを編集する。

## 別添のコメントアウト

`(別添N)`のみの行はコメントアウトする。

## 分類、名称を補う

`ke.txt`は以下の構造を持っていると想定されている。

```
ＡＡＡ関係              # 分類
【ＢＢＢ】              # 名称
問Ｎ   ＣＣＣ
       ＣＣＣ
（答） ＤＤＤ
       ＤＤＤ
問Ｎ   ＣＣＣ
       ＣＣＣ
（答） ＤＤＤ
       ＤＤＤ
【ＢＢＢ】              # 名称
問Ｎ   ＣＣＣ
  .....
```

分類は`関係`で終わる行であり、
名称は`【`及び`】`で囲まれた行である。
分類や名称が省略されている（欠けている）場合は
適宜、補う必要がある。

## 項番の編集

疑義解釈では問や答の中の文章に現れる項番（①、②やア、イ）などを
構文解析せず、そのまま`<pre>`、`</pre>で囲って
HTML出力する。
そのため、各行の長さやインデント位置に注意して編集する必要がある。

## 追加

疑義解釈の追加がある場合は直接`ke.txt`を編集してもよいが、
PDFファイルからのコピー・ペーストでは効率が悪い場合がある。
その場合は`in/NN/g/pdf/K`に追加のPDFファイルを配置し、
以下を実行する。

```
./run NN g0
```

`k.txt`に追加された末尾のテキストを
`ke.txt`の末尾にコピー・ペーストするとよい。


# HTMLの生成

編集したテキストファイル(ke.txt)
からHTMLを生成する。
ke.txtは分類、名称でソートしてからHTMLファイルを生成する。
このためke.txtとは順序が異なる点に注意する。

```
./run NN g2
```

様式はないのでステップ`g1`はない点に注意する。
