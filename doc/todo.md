## 施設基準の様式へのリンク

（2026-05-01 解決）

以下の「及び」以降がリンクにならない。
(施設基準(通知) 特掲診療料の施設基準等 第９ 在宅療養支援診療所 ４届け出に関する事項)

```
別添２の様式 11 及び様式11の３
```

以下の「、」以降がリンクにならない。

```
別添２の様式 11、様式11の３及び様式11 の５
```

リンク化しているのはrendererの施設基準通知本文クラス。
正規表現定義はPat内。

```
    public static final String 施設基準様式名パターン = "別\\s*添\\s*" + 数字
        + "\\s*の\\s*(別\\s*紙|様\\s*式)\\s*" + 数字 + "(\\s*の\\s*" + 数字 + ")*";
```
