# ディレクトリ構成

## 全体構成

```
プロジェクトホーム/
   入力/
      home/
         index.html          ホームページ
         hikaku.html         比較ページ
         all.css             共通CSS
         menu.js             メニュー作成JavaScript
      04/                    令和04年度
         i/                  医科
            pdf/             PDFはそれぞれ複数あってもよい。複数ある場合、ファイル名の辞書式順に処理する。
               k/            告示PDF
                   01.pdf    告示PDF(1)
                   02.pdf    告示PDF(2)
               t/            通知PDF
                   01.pdf    通知PDF(1)
               y/            様式PDF
                   01.pdf    様式PDF(1)
            txt/             テキスト
               k.txt         PDFから変換された告示テキスト
               ke.txt        編集された告示テキスト
               t.txt         PDFから変換された通知テキスト
               te.txt        編集された通知テキスト
               y.txt         PDFから変換された様式一覧
               ye.txt        編集された様式一覧
            img/        　    HTMLから参照するイメージ(様式は含めない)
               a.png 
         s/                  歯科(医科と同様のディレクトリ構成)
         t/                  調剤(医科と同様のディレクトリ構成)
         k/                  施設基準(医科と同様のディレクトリ構成)
   出力/
      index.html             ホームページ(入力/homeからのコピー)
      hikaku.html            比較ページ(入力/homeからのコピー)
      all.css                共通CSS(入力/homeからのコピー)
      menu.js                メニュー作成JavaScript(入力/homeからのコピー)
      04/                    令和04年度
         i/                  医科
            index.html       医科ホームページ
            yoshiki.html     医科様式一覧
            y/               医科様式
               x.pdf
            img/
               a.png         HTMLから参照するイメージ(入力/i/imgからのコピー)
         s/                  歯科(医科と同様のディレクトリ構成)
         t/                  調剤(医科と同様のディレクトリ構成)
         k/                  施設基準(医科と同様のディレクトリ構成)
```
