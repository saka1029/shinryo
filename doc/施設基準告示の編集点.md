# 施設基準通知の編集点
## 文書の構成

文書は少なくとも基本診療料の施設基準と特掲診療料のふたつのファイルに分かれている。
それぞれ本文と別表から構成される。
「別表」は様式ではないのでPDFではなくHTMLに変換する。

```
施設基準告示
    基本診療料(000602943.pdf)
        第一 届出の通則
        第二 施設基準の通則
        第三 初・再診料の施設基準等
        .....
        第十一 経過措置
        別表第一 地域歯科診療支援病院歯科初診料に係る手術
        .....
        別表第十五 特定入院料のみで届出可能な対象入院料
    特掲診療料(000602944.pdf)
        第一 届出の通則
        第二 施設基準の通則
        .....
        第十七 経過措置
        別表第一  特定疾患療養管理料並びに処方料並びに処方箋料の特定疾患処方管理加算１及び特定疾患
        .....
        別表第十三 広範囲顎骨支持型補綴及び広範囲顎骨支持型補綴物修理に規定する特定保険医療材料
```

## 括弧の誤り訂正

施設基準告示は縦書きPDFであるため、括弧が正しい位置に配置されていないことがある。
基本的には閉じ括弧が前の行に配置されているので、これを訂正する。

```
         ③   急性期一般入院料２の施設基準
           １) 次のいずれかに該当すること。
              (一  一般病棟用の重症度、医療・看護必要度Ⅰの基準を満たす患者を二割八分以上入院
              ) させる病棟であること。
              (二  診療内容に関するデータを適切に提出できる体制が整備された保険医療機関であっ
                て、一般病棟用の重症度、医療・看護必要度Ⅱの基準を満たす患者を二割六分以上入
                院させる病棟であること。
```

これは以下のように訂正する。

```
         ③   急性期一般入院料２の施設基準
           １ 次のいずれかに該当すること。
              (一)  一般病棟用の重症度、医療・看護必要度Ⅰの基準を満たす患者を二割八分以上入院
               させる病棟であること。
              (二)  診療内容に関するデータを適切に提出できる体制が整備された保険医療機関であっ
                て、一般病棟用の重症度、医療・看護必要度Ⅱの基準を満たす患者を二割六分以上入
                院させる病棟であること。
```
## 別表の行分割

別表はほとんどリストなので、各行が追い込まれないように"<br>"を適宜追加する。

```
    別表第一  特定疾患療養管理料並びに処方料並びに処方箋料の特定疾患処方管理加算１及び特定疾患
      処方管理加算２に規定する疾患<br>
        結核<br>
        悪性新生物<br>
        甲状腺障害<br>
        処置後甲状腺機能低下症<br>
        糖尿病<br>
        スフィンゴリピド代謝障害及びその他の脂質蓄積障害<br>
```

項番が振られている行はこの限りでない。

```
    別表第二  特定疾患治療管理料に規定する疾患等
      一  特定薬剤治療管理料１の対象患者
        (1)   テオフィリン製剤を投与している患者
        (2)   不整脈用剤を投与している患者
        (3)   ハロペリドール製剤又はブロムペリドール製剤を投与している患者
```
